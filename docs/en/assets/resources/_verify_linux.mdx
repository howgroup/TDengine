In the Linux shell, execute `taos` directly to connect to the TDengine service, entering the TDengine CLI interface, as shown below:

```text
$ taos

taos> show databases;
              name              |
=================================
 information_schema             |
 performance_schema             |
 db                             |
Query OK, 3 rows in database (0.019154s)

taos>
```
